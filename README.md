# Responsive Landing Page

This is a fully responsive landing page built using HTML, CSS, and JavaScript. It includes a hero section with heading, subtext, and a call-to-action button, a features section with three feature cards, a testimonials section, a pricing section, and a footer. The design is clean, modern, and works on all screen sizes. No frameworks such as Tailwind or Next.js were used.

## Features
- Responsive hero section with button
- Three feature boxes
- Testimonials section with user quotes
- Pricing section with highlighted plan
- Simple and clean footer
- Fully mobile responsive
- Written in pure HTML, CSS, and JavaScript

## Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla)

## Folder Structure
index.html  
style.css  
script.js  
README.md  

## How to Run
Download or clone the repository and open the index.html file in any browser. No installation or build tools required.
